项目背景

- 面向第一次养宠的用户，信息零散且难以判断可靠性，常见“不知道从哪开始查”的痛点
- 提供“能搜、能读、能留痕”的一体化入口：搜索直达百科，点开即阅读，登录后可持续积累
- 设计更注重人文关怀与新手友好，强调显著安全声明与清晰的信息层级，降低决策焦虑
核心功能

- 关键词搜索
  - 支持按 title 与 description 精准匹配
  - 无搜索时默认显示“热门百科”，提升首屏可用性
- 百科阅读
  - 搜索结果以卡片呈现，点击标题即可就地展开完整内容（description）
  - 标签与内容结构清晰，便于快速筛选与理解
- 用户登录
  - 基于 Supabase Auth 的邮箱密码登录/注册
  - 成功后自动在 profiles 表写入/更新用户资料
  - 登录状态即时刷新到导航栏显示“你好，[用户名]”，并支持退出
技术亮点

- 2026 主流的 AI 协作开发流程
  - AI 结对编程：从需求到实现的端到端闭环，持续“实现—预览—修正”循环
  - 任务驱动与可验证交付：将每次需求拆分为可验证的改动，保持高频小步迭代
  - 原型到工程的迁移路径清晰：先用 CDN 单文件快速跑通，再迁移到 Vite + React + Tailwind 的正式工程
- Supabase 实时后端
  - Auth：邮箱密码注册与登录、会话管理、用户资料写入（profiles upsert）
  - 数据库：Postgres + RLS（行级安全）策略，确保读写安全可控
  - Realtime 能力可扩展：后续可对 posts/wiki 开启订阅，支持社区内容与百科的实时更新
- 前端体验与可维护性
  - Tailwind 主题色与组件化结构，统一视觉与规范，易于扩展
  - Hash 路由与本地持久化（localStorage），在轻量环境下仍保证良好可用性
  - 就地展开的阅读交互，减少页面跳转与上下文切换，提升阅读效率
